# 四川东方巨擎工业技术有限公司 - 企业网站

这是一个基于 React + TypeScript + Vite 构建的现代化企业官网，采用 Geotab 网站的设计风格。

## 技术栈

- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **CSS3** - 样式设计

## 开发

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

构建生产版本：

```bash
npm run build
```

预览生产构建：

```bash
npm run preview
```

## 项目结构

```
src/
├── components/          # React 组件
│   ├── Header.tsx      # 导航栏
│   ├── Hero.tsx        # 主视觉区域
│   ├── About.tsx       # 企业简介
│   ├── Products.tsx    # 核心产品
│   ├── Advantages.tsx  # 技术优势
│   ├── Global.tsx      # 全球业务与使命
│   └── Footer.tsx      # 页脚
├── App.tsx             # 主应用组件
├── main.tsx            # 入口文件
└── index.css           # 全局样式
```

## 设计特点

- 现代化的企业设计风格
